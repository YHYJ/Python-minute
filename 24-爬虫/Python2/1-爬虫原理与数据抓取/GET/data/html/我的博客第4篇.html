git的使用

Git 2017年11月9日 20:33 YJ 0 评论 30 阅读

git提交代码更新到github：

配置git：

git config --global user.name "user_name"  # 主要开发者姓名
git config --global user.email "email"  # 邮箱

链接github仓库：

# 这里的 name 是你github账户名，repos_name 是你创建的仓库名
git remote add origin https://github.com/name/repos_name.git

第一次：

git init    # 初始化一个空仓库
git add .   # 把没有忽略的文件都添加到这个仓库

提交：

git commit -am "本次提交信息"
git push -u origin master    # 提交到仓库
